## Getting started
```sh
npm install --save react-native-basic-constants
```

## Example
```javascript
import BasicConstants from 'react-native-basic-constants';

render(){
  return (
    <Text style={{fontSize: BasicConstants.Style.H1}}>Some huge text</Text>
  )
}

```
